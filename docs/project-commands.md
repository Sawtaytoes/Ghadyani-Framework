# Project Commands

Default runner (same as yarn web:watch)
```shell
yarn start
```

Run the project
```shell
yarn web
```

Restart when files change
```shell
yarn web:watch
```

Compile bundles
```shell
yarn compile
```

Run web server with precompiled bundles
```shell
yarn server
```

Run unit tests
```shell
yarn test
```

Run unit tests and watch for changes
```shell
yarn test:watch
```
